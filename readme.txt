== Description ==

开发插件的初衷：

在推广的过程中，想必大家都有遇到过，推广链接失效了，却没有第一时间知道这个消息，从而可能带来了损失。

所以该功能点，基于此出发，提供自动检测链接是否失效的功能，并可以通过配置指定的文章进行评论，从而告知推广者链接失效的消息。

目前的失效提示的方式，有文章评论，后续可能会接入邮件，或者微信通知等。

目前检测链接是否失效的原理是，会先判断该链接服务器是否正常和通过后台配置的指定关键词来检测，这样的双重检测，可以更加准确的得知该链接是否失效，所以大家在添加活动的时候，尽量查看下所推广的链接的包含的关键词有哪些是很有必要的操作。同时这块的话，后续也会考虑检测链接是否在微信/QQ中可以正常打开。

基本操作：

1、在活动列表添加要推广的活动信息

2、添加完成后，点击调用代码按钮进行复制。对了，短代码大概是长这样子：[YrActivityLink id="我是id"]我是title[/YrActivityLink]，如果你需要修改字体颜色/大小，正确的使用方法:点我查看

3、将刚才第2步生成的短代码，粘贴到你想要的文章中即可。

4、快去前台预览效果吧

== Installation ==

1. 上传 `yr-activity-link`目录 到 `/wp-content/plugins/` 目录
2. 激活插件，开始设置使用。
